Frank's Gaming Café

Summary

Frank's Gaming Café will be able to support 24 players via a wired connection from the switch and more from wireless clients. The gaming systems are powerful enough to play almost all modern games in decent graphical settings. There will be a decent dedicated server machine to run and host the game servers. Games can be distributed to every computer very effectively, and the whole experience should be very good. In the end, most of the budget will be spent with several dollars left to spare.

Specifications

Computer Specifications

Peripheral Specifications

The Monitor

The Monitor is a BenQ RL2455HM 24in LED Gaming Monitor, which has a Full HD 1080p Panel with a 1ms GTG response time. This means that there will be less ghosting or motion blur, providing for a more seamless gaming experience. It also comes at a good budget price of $200 for 24inches of diagonal screen real estate.

The Keyboard

The keyboard is a mechanical keyboard, which means it has better tactile feedback than membrane keyboards. Mechanical keyboard also last for longer than membrane keyboard and is guaranteed to last up to 60 million keystrokes. This keyboard also has fully programmable macro keys for gaming and comes packaged with Razor's Synapse software for controlling certain features of the keyboard.

The Mouse & Mouse Pad

The mouse is has an optical sensor and has the ability to change the DPI of the mouse on the fly for easier gaming. The mouse pad has the largest surface area for the very cheap price and will make the experience of using an optical mouse better due to having a set pattern and being made of a soft material. The mouse pad makes the sensor more accurate and more comfortable.

Gaming PC Specifications

Each of the Gaming PCs in this LAN Party setup will be custom built with the components above. The CPU will be overclocked and cooled using a closed loop water cooling system with the radiator mounted to the back exhaust of the case. It is a dual core CPU, which has enough cores for most modern games today, and having extra cores may not affect the playability of the game. The motherboard has the LGA1150 Z97 Chipset, which supports overclocking and had enough PCI-E 16x lanes to run two graphics cards. It also has built in networking, which eliminates the need for a dicrete network card. There is also a GTX 780 in each machine, providing more than adequate graphical horsepower to render the frames in 1080p. The two SSDs will be put in RAID 0 to provide even faster speeds to load games faster and to reduce any storage related bottlenecks. There is also enough RAM with a low latency to run games smoothly. The Power Supply is 80 Plus Bronze rated to provide enough contant power. The case is a Mid-Tower by Coolermaster which is both cheap, and provides adequate cooling and space for the system.

Server System Specifications

This server computer is built for mainly server use and does not have a dicrete graphics card for gaming. It shares many similar components to the Gaming System, but instead, has 4 SSDs to run in RAID 0 for even greater speed, two 4TB HDDs for game and media storage, a quad core CPU which will be overclocked for running the game server software, a total of 32GB of high speed RAM, a PCI-E network card, and an optical drive for loading any media from disks.
The reasoning behind the quad core CPU, is that server software will mostlikely be multi-threaded, and having more cores will likely be beneficial, on top of this, it will be overclocked to provide even more performance. There will be 4 SSDs running in RAID 0 for more storage and speed. Then the two 4TB HDDs will be used for storing games and other software. This is because the server system will also act as shared storage and/or a NAS (Network Attached Storage). This means that a game will be loaded onto the server computer and be accessible by the gaming systems. The discrete network card will be used to provide better, faster networking performance for file transfer and LAN gaming. The increased amount of RAM will aid the performance of any server software and allow for a more seamless multiplayer experience. By having an optical drive on the server computer, games can be copied to the server computer, then to the gaming systems more efficiently, rather than manually copying games to each gaming system.

Network Specifications

The router is one of the most advanced Wi-Fi routers. The Linksys WRT1900AC Dual Band Wireless AC 1900 Router has the top of the line performance in networking, and its very strong Dual Band 2.4 and 5 GHz wireless AC means that laptops can join the LAN Party. Connected to the router, is a 24 port switch by TP-Link which supports Gigabit networking. To connect the gaming systems and the server system together, will be 5 metre Cat6 RJ-45 (Ethernet) cables for maximum through put. The server system and the switch will plug directly into the router.

Other Specifications

Power

Power will be provided by three 10 port powerboards and 5 meter extension cables with a mains double adaptor at the end to power the computer and the monitor.
